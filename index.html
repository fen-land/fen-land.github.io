Fen

Simple Framework To Create Server on deno

Github

Introduction

Fen is a simple web framework that help you to build web server.

It's powerful with tool and process, which will be more in later version.

Help us by join us or commit issues on github.

How to use

First, you need install deno:

curl -fsSL https://deno.land/x/install/install.sh | sh

Second, import server from github:

import { Server } from "https://github.com/fen-land/deno-fen/raw/master/server.ts";
const s = new Server();
s.start();

Finally, run script in the command line:

deno ${yours}.ts -A
